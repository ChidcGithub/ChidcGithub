## Mobile Development

My most recent project is **CountDown**, a Flutter-based Android application that generates unique death countdown timers based on user input. Built with Material Design 3, this entertainment app demonstrates my skills in Flutter widget development, state management, and local data persistence using SharedPreferences. I also developed **Fishnet**, another Flutter application designed for testing and recording various types of errors, which reflects my focus on building robust and debuggable applications.

## Machine Learning & AI

I have a strong interest in artificial intelligence and machine learning. **MLSharp-3D-Maker-GPU** is a sophisticated project that provides 3D Gaussian splatting generation based on the Apple Sharp model. This tool supports GPU inference on NVIDIA, AMD, and Snapdragon devices, with ongoing work to adapt it for Snapdragon's NPU for faster processing. This project highlights my abilities in Python, deep learning frameworks, and GPU computing.
